# Supermarket Platform - Client Guide

Welcome to your new supermarket platform! This guide will help you manage your online store, add products, process orders, and customize your branding.

## Getting Started

### First Login

1. Open your web browser
2. Go to your store URL (e.g., `https://yourstore.com`)
3. Click "Admin Login" or go to `https://yourstore.com/admin`
4. Enter your admin credentials:
   - Email: `admin@yourstore.com`
   - Password: (provided separately)

### Dashboard Overview

Your admin dashboard shows:
- **Sales Overview**: Today's revenue and orders
- **Quick Stats**: Total products, customers, orders
- **Recent Orders**: Latest customer orders
- **Low Stock Alert**: Products that need restocking

## Managing Products

### Adding Product Categories

1. Go to **Products → Categories** in the admin menu
2. Click "Add New Category"
3. Fill in the details:
   - **Category Name**: e.g., "Fresh Vegetables"
   - **Description**: Brief description for customers
   - **Upload Image**: Add a category photo
4. Click "Save Category"

### Adding Products

1. Go to **Products → All Products**
2. Click "Add New Product"
3. Fill in product information:

#### Basic Information
- **Product Name**: e.g., "Organic Tomatoes"
- **Description**: Detailed description for customers
- **Category**: Select the appropriate category
- **SKU**: Unique product code (optional)

#### Pricing
- **Regular Price**: Standard selling price
- **Sale Price**: Discounted price (optional)
- **Cost Price**: Your purchase cost (for profit tracking)

#### Inventory
- **Stock Quantity**: How many items you have
- **Low Stock Alert**: Get notified when stock is low
- **Track Inventory**: Enable stock tracking

#### Images
- **Product Photos**: Upload 1-5 high-quality images
- **Image Guidelines**:
  - Use clear, well-lit photos
  - Show product from multiple angles
  - Include size reference (like a coin or ruler)
  - Minimum 800x800 pixels

4. Click "Save Product"

### Managing Product Inventory

1. Go to **Products → Inventory**
2. Update stock quantities:
   - Enter new quantities manually
   - Or use bulk update for multiple products
3. Set up low stock alerts
4. Click "Update Inventory"

## Managing Promotions

### Creating Discounts

1. Go to **Marketing → Promotions**
2. Click "Create New Promotion"
3. Set promotion details:
   - **Promotion Name**: e.g., "Weekend Sale"
   - **Discount Type**: Percentage or fixed amount
   - **Discount Value**: e.g., 20% or $5 off
   - **Applicable Products**: Choose specific products or categories
   - **Start/End Dates**: When the promotion runs
4. Click "Create Promotion"

### Special Offers

- **Bundle Deals**: Combine multiple products for a discount
- **Buy One Get One**: BOGO promotions
- **Free Shipping**: Offer free delivery on minimum orders

## Processing Orders

### Viewing Orders

1. Go to **Orders → All Orders**
2. Orders are automatically categorized:
   - **New**: Orders awaiting processing
   - **Processing**: Orders being prepared
   - **Ready**: Orders ready for pickup/delivery
   - **Completed**: Orders delivered
   - **Cancelled**: Cancelled orders

### Processing an Order

1. Click on an order to view details
2. Review customer information and items ordered
3. Update order status:
   - **Confirm**: Verify order details
   - **Process**: Start preparing the order
   - **Ready**: Mark as ready for delivery
   - **Complete**: Confirm delivery

### Order Details Include

- **Customer Information**: Name, phone, delivery address
- **Order Items**: Products, quantities, prices
- **Payment Status**: Paid, pending, failed
- **Delivery Information**: Preferred time, special instructions

#### Colors & Theme

## Troubleshooting

### Common Issues

**Products Not Showing**
1. Check if product is marked "Active"
2. Verify stock quantity > 0
3. Ensure category is active

**Orders Not Processing**
1. Check payment status
2. Verify customer information
3. Check inventory availability

**Images Not Uploading**
1. Check file size (max 5MB)
2. Verify file format (JPG, PNG, WebP)
3. Check internet connection

**Customers Can't Checkout**
1. Verify delivery settings
2. Check payment gateway status
3. Ensure store is not in maintenance mode
